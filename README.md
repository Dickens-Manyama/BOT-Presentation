# BOT presentation

Group presentation slides for the Bank of Tanzania automated data collection and validation pipeline.

## Viewing

Open `index.html` in a browser (or deploy the folder to GitHub Pages so the site loads at `/`). Use the **Presentation** control for fullscreen navigation (arrow keys, Prev / Next).

## Slides

Seventeen HTML files: `index.html` (title) plus `slide-1.html` through `slide-16.html`, shared assets, and PNG figures used by the deck.
